# The Maze

The Maze is a 3D Maze game that uses ray casting to render a 2D map into a 3D navigable world!

The Maze was written in C using SDL2 library. 

## Instalation 
```sh
$ git clone https://github.com/TomLutherKE/maze_project.git
```
## Usage 
* Execute ./maze or type make run 
* Use up and down arrow keys or "W" and "S" keys to move forward and backward respectively.
* Use right and left arrow keys or "D" and "A" keys to turn the camera clockwise or counterclockwise respectively.
